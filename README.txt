OBS: Para executar o chat precisa ter as 
dependencias do node instaladas na máquina.

1- Abrir o prompt de comando
2- Colocar no mesmo diretorio da pasta do chat
pode ser utilizado o comando 'cd' e o 
caminho do arquivo para isso
3- Executar o comando: node server.js
4- Acessar o localhost:1998 já estará funcionando
